# Lylo pilot docs

Onboarding, demo guides, and operational documentation for the
people running a Lylo pilot deployment.

## What lives here (target state)

- `onboarding-guide.md` - how an operator creates a new pilot
  instance from scratch. Uses the `scripts/create-pilot-instance.js`
  CLI shipped in PR G.
- `security-overview.md` - one-page summary suitable for a pilot
  partner or reviewer (nonprofit, senior center, family). Plain
  language; no jargon.
- `demo-walkthrough.md` - the script for a demo session: what to
  show, what to say, what *not* to claim.
- `setup-wizard-guide.md` - end-user guide for the Setup Mode
  wizard, written for the family member or caregiver who is
  walking the senior through onboarding.

## Audience

The writing in this directory is for non-engineers: nonprofit
staff, senior-center coordinators, caregivers, family members.
Keep the language plain, avoid jargon, and never use the language
banned by the Phase 1 audit (`consciousness`, `oracle`, `soul`,
etc.).
